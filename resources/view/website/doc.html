O que é JavaScript?

É uma linguagem de programação criada inicialmente para dar vida a páginas da web estáticas, é uma linguagem interpretada de tipagem fraca e multiparadigma. Tipagem fraca porque as variáveis ​​declaradas serão capazes de assumir valores de diferentes tipos de dados e realizar operações entre eles sem a necessidade de uma conversão. Multiparadigma por suportar vários paradigmas de programação, como programação funcional ou orientada a objetos.

Apesar de seu início focado no desenvolvimento web, a linguagem ao longo dos anos foi se atualizando e hoje está presente na criação de aplicativos móveis nativos utilizando um framework como o React Native, no desenvolvimento de aplicativos desktop utilizando Electron, no Back-end de aplicativos com Node.Js e até mesmo criando inteligência artificial usando TensorFlow.

Agora que você tem uma pequena ideia do que é JavaScript, você já sabe onde ele está presente e seu potencial, vamos da início aos códigos.

Variáveis, VAR, LET, CONST, o que são e como criar?

Em qualquer linguagem de programação, as variáveis ​​são uma forma de reservar um espaço na memória para armazenar e reutilizar dados durante o uso do aplicativo. Com o tempo, você descobrirá que as variáveis ​​podem conter mais do que apenas dados, por exemplo, podem conter funções.


// Existem 3 maneiras de declarar uma variável JavaScript:

var x = 1;
let y = "2";
const xy = "terceira forma";

A primeira forma de criar uma variável é usando VAR, esta também é a forma mais antiga da linguagem.


// Quando usado dessa forma, a variável acaba tendo seu escopo global.

var x = 1;

A segunda forma é usando LET, que foi implementada a partir da versão ES2015.


// LET é usado dentro de uma função JavaScript, o que permite que seu escopo seja local.

var xy = 10;

function functionTest(x,y) {
let xy = 5;
return x + y - xy;
}

console.log(functionTest(4,3));

Observe que XY foi usado com VAR e XY com LET, mas como mencionado antes, LET tem escopo local, então o resultado será 2 em vez de -3, isso acontece porque literalmente duas variáveis ​​foram criadas, com valores diferentes, porém como LET XY foi definido dentro da função, o javascript entenderá que deve usar a variável de escopo local.


//E finalmente a variável CONST

const xy = "terceira forma";

CONST tem comportamento semelhante a LET, a diferença é que seu valor não pode ser trocado por uma atribuição. Também tem escopo global, ou seja, ao declarar uma variável CONST fora da função, você terá acesso ao seu valor dentro da função, o que não acontece com LET.

Ao nomear variáveis ​​você deve seguir algumas regras, caso você não siga essas regras, ocorrerá um erro em sua aplicação.

  • Os Nomes podem conter letras, dígitos, sublinhados e cifrões.
  • OS Nomes não podem iniciar com números.
  • Os nomes são sensíveis a casos (y e Y são variáveis diferentes).
  • As Palavras reservadas não podem ser usadas como nomes.
